What is CSS?

CSS(Cascading Style Sheets) is the code that styles web content. Like HTML, CSS is not a programming language. It's not a markup language either. CSS is a style sheet language. CSS is what you use to selectively style HTML elements. For example, this CSS selects paragraph text, setting the color to red: 

      
        p {
          color: red;
        }

Why Use CSS?

CSS is used to style web pages. Without CSS, web pages would all look the same. With CSS, you can change colors, add animations, and create layouts. CSS is a powerful tool for web designers.

How to Use CSS

There are three ways to use CSS: inline, internal, and external. Inline CSS is used to style a single HTML element. Internal CSS is used to style a single web page. External CSS is used to style multiple web pages.

Anatomy of a CSS ruleset

A CSS ruleset consists of a selector and a declaration block. The selector is the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon.

Different types of selectors

There are many different types of selectors in CSS.

Types of Selectors
Selector Description
Element Selector Selects all instances of an HTML element
ID Selector Selects a single HTML element with a specific ID
Class Selector Selects all HTML elements with a specific class
Descendant Selector Selects an HTML element that is a descendant of another HTML element
Child Selector Selects an HTML element that is a child of another HTML element
Adjacent Sibling Selector Selects an HTML element that is the adjacent sibling of another HTML element
Attribute Selector Selects an HTML element with a specific attribute
Pseudo-class Selector Selects an HTML element with a specific state
Pseudo-element Selector Selects a specific part of an HTML element
